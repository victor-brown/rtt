# Build:

`docker build -t php-web-app .`

# Run:

`docker run -p 80:80 php-web-app`

# Open:

`http://localhost`

# Cypress:

`cd cypress`
`npm i`
`npx cypress run --spec "cypress/e2e/example_spec.cy.js"`
